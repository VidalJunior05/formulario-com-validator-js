## 📁 How to download the Project 
```bash 
    #clone the repository
     $ git clone https://github.com/VidalJunior05/formulario-com-validator-js.git

    # Enter directory
    $ cd validador-de-formulario

    # install dependencies
    $ yarn install

    # Initialize project
    $ yarn start
```
